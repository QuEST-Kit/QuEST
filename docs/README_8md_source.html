10 ## Quick Start
11 
12 Copy or clone this repository to your machine. E.g. in the desired directory, enter
13 ```bash
14 git clone https://github.com/TysonRayJones/QuEST.git
15 ```
16 at terminal.
17 
18 In the root directory, compile the [example](examples/tutorial_example.c) using
19 ```bash
20 cp examples/tutorial_example tutorial_example
21 make
22 ```
23 then run it with
24 ```bash
25 ./demo
26 ```
27 
28 The program will print information about your execution environment and some simple operations on a three qubit system. See the [tutorial](examples/README.md) for a better introduction. Additionally, run `tests/runTests.sh` to test QuEST runs correctly in your environment.

31 ## Introduction
32 
33 The **Quantum Exact Simulation Toolkit** is a high performance simulator of universal quantum circuits. QuEST is written in C, hybridises OpenMP and MPI, and can run on a GPU. Needing only compilation, QuEST is easy to run both on laptops and supercomputers, where it can take advantage of multicore, GPU-accelerated and networked machines to quickly simulate circuits on many qubits.
34 
35 QuEST has a simple interface, independent of its run environment (on CPUs, GPUs or over networks),
36 ```C
37 hadamard(qubits, 0);
38 
39 controlledNot(qubits, 0, 1);
40 
41 rotateY(qubits, 0, .1);
42 ```
43 though is flexible
44 ```C
45 Vector v;
46 v.x = 1; v.y = 0; v.z = 0;
47 rotateAroundAxis(qubits, 0, 3.14/2, v);
48 ```
49 and powerful
50 ```C
51 // sqrt(X) with pi/4 global phase
52 ComplexMatrix2 u;
53 u.r0c0 = (Complex) {.real=.5, .imag= .5};
54 u.r0c1 = (Complex) {.real=.5, .imag=-.5};
55 u.r1c0 = (Complex) {.real=.5, .imag=-.5};
56 u.r1c1 = (Complex) {.real=.5, .imag= .5};
57 unitary(qubits, 0, u);
58 
59 int[] controls = {1, 2, 3, 4, 5};
60 multiControlledUnitary(qureg, controls, 5, 0, u);
61 ```

63 ## Getting started
64 
65 QuEST is contained entirely in the files in the `QuEST/` folder. To use QuEST, copy this folder to your computer and include `QuEST.h` in your `C` or `C++` code, and compile using the [makefile](makefile). We include [submission scripts](examples/submissionScripts/) for using QuEST with SLURM and PBS. See the [tutorial](/examples/README.md) for an introduction.
66 
67 Explicit instructions to download and run QuEST from the command line can be found at [quest.qtechtheory.org/download](https://quest.qtechtheory.org/download/).
