Rotate a single qubit in the state vector of probability amplitudes, given the angle rotation arguments and a control qubit.

Only perform the rotation for elements where the control qubit is 1. alphaRe = cos(angle1) * cos(angle2)
alphaIm = cos(angle1) * sin(angle2)
betaRe = sin(angle1) * cos(angle3)
betaIm = sin(angle1) * sin(angle3)

Remarks
Qubits are zero-based and the the first qubit is the rightmost

Measure in Zero performs an irreversible change to the state vector: it updates the vector according to the event that an outcome has been measured on the qubit indicated by measureQubit (where his label starts from 0, of course).
